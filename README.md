## leafage-basic

**leafage-basic 包含两个模块「assets」和 「hypervisor」**

#### 「assets」目标功能

- [x] 帖子
- [x] 影视集
- [x] 分类
- [x] 访问统计
- [ ] 点赞

#### 「hypervisor」目标功能
- [x] 用户、角色、权限
- [x] 账户管理
