## leafage-basic

**leafage-basic 包含两个模块「assets」和 「hypervisor」**

#### 「assets」目标功能
- [x] 博客
- [x] 作品集 
- [x] 资源统计
- [ ] 点赞、留言、评论
- [ ] 分类、标签、归档

#### 「hypervisor」目标功能
- [x] 用户、角色、权限
- [x] 账户管理 
- [x] 安全
