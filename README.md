## abeille-basic

**abeille-basic 包含两个模块「abeille-basic-assets」和 「abeille-basic-hypervisor」**

「abeille-basic-assets」是分享的博客、资源管理的功能；

「abeille-basic-hypervisor」是基础功能，主要提供权限，用户等功能；
